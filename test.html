엑셀(.xlsx, .xls) 파일을 업로드하세요.
파일을 이 영역으로 드래그&드롭하거나, 상단의 파일 업로드 버튼을 사용하세요.

기기(일련번호) 관리

활성 기기: 없음
쉼표/공백으로 여러 개 입력 가능. 끝의 마침표/쉼표는 자동 제거됩니다.
부품 사용 상태
XLSX 메모 태그
기기 고장증상 0/290 바이트
활성 기기를 선택하면 고장증상을 입력할 수 있습니다.

태그 관리

파일 가져오기 (L열 자동 인식)

미리보기 (L열, 2행부터, 최대 10행)

태그 → 카테고리 매핑

발견된 태그들을 자전거, 킥보드, 공용으로 분류해주세요.

부품 리스트

선택된 항목(현재 기기 메모)

참고: 메모(현재 기기 출력)는 내부 공백이 제거됩니다. "장력조정" 선택 시 메모에는 "장력조정"이 표시됩니다.
